KRX 퀀트 리서치 프레임워크
Quantocracy 글로벌 아이디어를 한국 주식시장에 적용하기 위한 표준 연구 프레임워크입니다.
1단계 — 아이디어 검증
  • 전략의 핵심 알파 소스 식별 (모멘텀/밸류/퀄리티/매크로)
  • 원 논문의 백테스트 기간, 유니버스, 리밸런싱 주기 확인
  • 비용(TC, 슬리피지) 가정 확인 및 KRX 기준 재조정
  • KRX에서 유사 팩터 선행연구 탐색
2단계 — 데이터 소스
  • 가격/거래량: KRX 정보데이터시스템 (data.krx.co.kr)
  • 재무데이터: FnGuide, Quantiwise, TS2000
  • 매크로: 한국은행 ECOS API (ecos.bok.or.kr)
  • 대안데이터: DART 전자공시 Open API
import FinanceDataReader as fdr import pykrx.stock as stock df = fdr.DataReader('KS11', '2000-01-01') uni = fdr.StockListing('KOSPI')
3단계 — KRX 특수 고려사항
  • 가격제한폭 ±30% — 극단값 처리
  • 개인투자자 비중 ~60% → 단기 과반응 경향
  • 재벌 구조 — 지주회사 할인 / 그룹사 상관관계
  • 공매도 제한 (주기적 금지) → Long-only 기준 병행
  • 거래세 0.18~0.35% → 회전율 높은 전략에 큰 영향
4단계 — 백테스트 지표
# 목표 지표 CAGR > KOSPI TR, Sharpe > 0.7 MDD < -25%, Calmar > 0.5 # 필수 서브기간 2008(금융위기), 2020(코로나), 2022(금리급등)
